# Innomatics_GenAi
🚀 Prompt Engineering using LangChain

This project demonstrates how to build dynamic and reusable prompt systems using LangChain instead of hardcoded prompts.

🔹 Key Features
PromptTemplate for reusable prompts
Multi-input prompt handling
Prompt variations (teaching, interview, storytelling)
ChatPromptTemplate for role-based interaction
Input validation system
Dynamic prompt generator
🧠 Tech Stack
Python
LangChain
📌 Outcome

Built a mini prompt engine to generate structured and scalable prompts for LLM-based applications.
